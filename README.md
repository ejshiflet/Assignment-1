# Assignment-1

Purpose: Generate passwords based on the user's choices and it can create a random list of 1000 passwords.

How to Use: First the main menu will display allowing you to select from generating a memorable password, a random password, 1000 passwords, or just exit the program. When you choose to make a memorable or random password, simply answer the questions on password preferences. Then the password will be displayed and written to a .txt file in the Memorable or Random folders. If the 1000 password option is chosen, it will generate a .txt file with 1000 passwords and will be in the ManyPasswords folder. All of these option return you to the main menu and you can choose the last option to exit the program.

Modules Used: random, os, string, time
